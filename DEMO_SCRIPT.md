# Demo script (for Trupeer video + science fair) — ~3 min

**Setup:** `python -m rescue.pipeline` (show the 4 agents run), then `python -m rescue.web` open in browser.

1. **The stakes (15s):** "Harven is 4 days from a regulatory audit and their warehouse data can't
   be trusted. Our user is the compliance officer — who has never opened a database."
2. **Run the agents (30s):** show the terminal: Agent 1 Find → 285 issues, Agent 2 Rank →
   99 critical, Agent 3 Act → 130 auto-fixable, Agent 4 Explain → signed summary. Point out each
   reads the previous from the shared memory layer.
3. **The product, used cold (60s):** open the web worklist. "Worst first. Every row says what's
   wrong, in plain English, and what to do." Click a filter (critical / auto-fixable). Read one
   row aloud — e.g. "customer CX-A228 isn't in the customer master → escalate."
4. **Explainability (20s):** "No row says 'the model said so' — every one has a concrete reason."
5. **The deliverable (20s):** click **Download signed summary** → show the one-page audit memo the
   officer can sign and hand to auditors.
6. **Close (15s):** "Find, fix, explain — broken data made auditable for someone who can't read SQL."

**Don't forget (mandatory):** Trupeer video URL, Geodo research (Domain Expert, on a few real
customers/plants), Product Brief PDF, GitHub link, Devpost submission by 5:00 PM.
